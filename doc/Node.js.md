# 部署步骤

- ## 必要文件  

| 文件或文件夹 | 内容 |
| -: | :- |
bin/www | 实际入口
public/images | 公共图片
public/javascript | 公共脚本
public/stylesheets | 公共样式
routes/* | 路由API
views/* | 页面模板
app.js | 入口文件
package.json | 项目配置文件

- ## 操作命令

> ``` shell
> cd projectPath
> npm install
> npm start
> ```


# 测试环境
